# FASDD-CV Project Page

Static academic project page for the FASDD-CV fire & smoke detection project.
Deployed via GitHub Pages.

## Deploying

See `DEPLOY.md` at the root of this repository.

## Editing links

Open `index.html` and update the four badge `href` attributes:
- **Report** → link to your PDF report
- **GitHub** → link to your code repository  
- **Demo** → link to your HuggingFace Space
- **HF Space** → same as Demo

Search for `href="#"` to find all placeholder links.
